Minecraft Game Review

The good and the bad.

The Good

The good thing about this game is that if you're in creative mode you can build some really cool stuff.

The Bad

The bad thing about this game is that sometimes if you're in survival mode a bunch of mobs can attack you at the same time on your first couple of nights and you can die.

Final Verdict

All in all this is a pretty fun game to play.
